# Démo — Espace apprenant Motival

Maquette cliquable de l'application Motival : deux parcours lancés depuis une page de garde.

- **Espace apprenant** : Connexion → Accueil → Espace formations → Visionneuse de support
- **Back-office** : Tableau de bord → Apprenants

## Lancer en local
Ouvrez `index.html` (ou `Demo Motival.dc.html`) dans un navigateur. Gardez tous les fichiers ensemble (`support.js` et `assets/` doivent rester à côté des pages).

## Mettre en ligne avec GitHub Pages
1. Créez un dépôt (ex. `demo-motival`) et déposez tout le contenu de ce dossier à la racine.
2. **Settings → Pages** → Source : branche `main`, dossier `/ (root)` → **Save**.
3. L'URL publique sera : `https://VOTRE-USER.github.io/demo-motival/`
   (la page d'accueil redirige automatiquement vers la démo).

> Une connexion internet est requise pour la police Lexend (chargée depuis Google Fonts).
